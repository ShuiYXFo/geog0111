Installation of these notes on UCL JupyterHub

  1. Make sure you have the UCL VPN installed and running
  2. Log on the the UCL JupyterHub.

  3. In a shell, type:

    /shared/groups/jrole001/geog0111/init/init0111.sh > init0111.log

This will clone this repository and set up all files and links you need.

Running on UCL JupyterHub

  1. Make sure you have the UCL VPN installed and running OR that you are running from Desktop@UCL
  2. Log on the the UCL JupyterHub.
  3. Navigate to the directory geog0111/notebooks
  4. Access the notebooks you want directly, or via the index.
  5. See further information on the course Moodle page
